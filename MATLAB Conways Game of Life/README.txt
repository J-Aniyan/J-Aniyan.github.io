MATLAB CONWAYS GAME OF LIFE

This is a final year project from my first year of the BSc Physics course at NTU. It is the famous Game of Life reproduced on MATLAB.
Users can start with a random initial grid preset, or investigate known configurations/features of the game.

Included are:
- Design.docx
This details the design and psuedocode of the project.
-GameOfLife.m
The MATLAB code. This will require an official MATLAB licence to run.
-Video Demonstration2.mp4
This is a video demonstration showcasing the project code, testing and a walkthrough of how to use it.
